# Integrador
Calculadora de integrais definidas por soma de Riemann

O software atualmente calcula áreas definidas por uma função implementando o método da Integral e Soma de Riemann. Utilizando aproximação pela "esquerda" e pela "direita", é possível obter o ponto médio da linha que define a função, aumentando, consequentemente, a precisão do resultado. 

Atualmente, novas versões estão sendo desenvolvidas para abarcar diferentes métodos de cálculo de área, além de melhorias na experiência do usuário, como uma interface gráfica simplificada, sem necessidade de uso do terminal ou edição do código fonte.
